## lazy function circle
### Arguments:
- center (`mcposition`)
- radius (`float`)
- angular_speed (`int`)
- action (`int=>void`)
### Return:
- void


Execute an action at each point of a circle of a given radius and a given center.

## lazy function line
### Arguments:
- start (`mcposition`)
- end (`mcposition`)
- step (`float`)
- action (`int=>void`)
### Return:
- void


Execute an action at each point of a line from start to end.
